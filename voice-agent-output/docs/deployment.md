# Deployment Guide: enterprise-grade-real-time

- Azure CLI >= 2.55.0
- Bicep CLI >= 0.24.0
- Docker >= 24.0
- Python >= 3.11
- GitHub CLI (optional)
- An Azure subscription with Contributor access

## Infrastructure Deployment

### Step 1: Login to Azure

```bash
az login
az account set --subscription <your-subscription-id>
```

### Step 2: Create Resource Group

```bash
az group create \
  --name rg-enterprise-grade-real-time-dev \
  --location eastus2 \
  --tags project=enterprise-grade-real-time environment=dev
```

### Step 3: Deploy Infrastructure

```bash
az deployment group create \
  --resource-group rg-enterprise-grade-real-time-dev \
  --template-file infra/bicep/main.bicep \
  --parameters infra/bicep/parameters/dev.parameters.json \
  --verbose
```

### Step 4: Build and Push Container

```bash
# Get ACR login server
ACR_NAME=$(echo "enterprise-grade-real-time" | tr -d '-')acr
ACR_LOGIN=$(az acr show --name $ACR_NAME --query loginServer --output tsv)

# Login to ACR
az acr login --name $ACR_NAME

# Build and push
docker build -t $ACR_LOGIN/enterprise-grade-real-time:latest -f src/app/Dockerfile src/app/
docker push $ACR_LOGIN/enterprise-grade-real-time:latest
```

### Step 5: Update Container App

```bash
az containerapp update \
  --name enterprise-grade-real-time \
  --resource-group rg-enterprise-grade-real-time-dev \
  --image $ACR_LOGIN/enterprise-grade-real-time:latest
```

### Step 6: Verify Deployment

```bash
# Get the FQDN
FQDN=$(az containerapp show \
  --name enterprise-grade-real-time \
  --resource-group rg-enterprise-grade-real-time-dev \
  --query properties.configuration.ingress.fqdn \
  --output tsv)

# Test health endpoint
curl https://$FQDN/health
```

  --name enterprise-grade-real-time \
  --resource-group rg-enterprise-grade-real-time-dev \
  --output table

  --name enterprise-grade-real-time \
  --resource-group rg-enterprise-grade-real-time-dev \
